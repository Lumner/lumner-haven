    Markdown Learning¶

    Lumner¶

    基本语法¶

    1. 标题
      # 一级标题
      ## 二级标题
      最多有六级标题，记得在#和标题名之间加空格
    2. 引用
      > 嘿嘿，真好玩
      > 这个不用加空格
      >> 这个引用还可以嵌套哟，嘿嘿
      > 在二级引用下再放一级引用会并入二级引用，回车后又比较难看
    3. 列表
      1. 无序列表
        • 在前面加-、+、*, 记得加空格
      2. 有序列表
        >我现在用的这些有标号的其实就是有序列表
      3. 嵌套
        1. 如果我在标号后不加空格就会被嵌套
        2. 然后其实上面的那些即便有空格也是在嵌套，我猜应该要这一级有空格，才能在下一级没有空格的时候嵌套进去
      4. todolist
        • 这个挺好玩的，不过-和[]之间，[]内部，[]和内容之间都要加空格
        • +、*也都可以达成这一效果

    4. 表格

      靠左 居中 靠右
      中间不用空格 横线个数随便 会自己根据字数调整表格大小

    5. 段落

      • 换行：两个以上空格后回车或者空一行
      • 分割线：三个及以上*、-、_（可以空格也可以不空）

      • 字体

        字体 代码
        斜体，斜体 * *, _ _
        高亮 == ==
        粗体，粗体 ** **,__ __
        斜粗体，斜粗体 *** ***,___ ___
        删除 ~~ ~~
        下划线 <u> </u>

      • 脚注
        脚注好好玩[^1]，用[^数字]的形式，然后要在结尾以[^1]:解释去说明一下
        [^1]: 嘿嘿

    6. 代码

      1. 一句代码可以用两个反引号给括起来: printf("hello world");
      2. 多句代码则可以用三个反引号给括起来:
        #include<stdio.h>  
int main()  
{  
printf("hello world");  
return 0;  
}

    7. 超链接

      • 更多使用教程可参考网站
      • 查看更多使用教程请点击链接

    8. 图片

      • 使用图床保存图片并实现插入(目前注册不了路过图床)
      • 路过图床
      • 使用 markdown 语法插入：
      • ![替代文字](图片路径)
      • ![替代文字](图片路径 "图片标题")
      • 等有类似于过路图床的软件或网站再说吧
      • 使用 html 语言实现调整图片大小和位置功能

    其他操作¶

    • 插入 latex 公式

      • 行内显示公式： \(f(x)=ax+b\)
      • 块内显示公式:
      \[ \begin{Bmatrix} a & b \\ c & d \end{Bmatrix} \]

    • html/css 语法

      • ctrl+shift+p 搜索 "Markdown Preview Enhanced:Customize CSS" 在 style 中使用 css 语法改标题格式等
    • 个性化设置
      • File -> Preferences -> Settings

    导出为 PDF¶

    • 用 Markdown PDF（下了拓展之后右键即可调出来）
    • 但是如果有数学公式的话还是不太推荐用这个
    • 右键 -> Open in Browser -> 在网页中另存为 PDF（可是我的浏览器就只能存为网页，也是很奇怪了）
